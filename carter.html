Carter

Carter is a shopping app for iOS designed around store profiles. Each store has its own profile page to make it easy for users to navigate between them. Users can add items and groups of items (collections) to their checkout and start an order.
The app is built using the MVVM design pattern, utilising frameworks such as SwiftUI, Combine, CoreAnimation. The sample data is mocked from a simple backend built using Vapor.
